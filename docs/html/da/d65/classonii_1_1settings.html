onii::settings::settings ( format  file_format)
inline

Constructor.

Parameters
[in]file_format- the file format to use

INI file format:

; All at the right of ';' or '#' is a comment
Key1=Value1
[Group1]
Key2=Value2
ListKey3[]=Value3
ListKey3[]=Value4

Will become after reading (Key / Value):

Key1 / Value1
Group1.Key2 / Value2
Group1.ListKey3[0] / Value3
Group1.ListKey3[1] / Value4

JSON file format:
